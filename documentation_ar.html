1. مقدمة

أداة المناطق السعودية المحسنة هي مكتبة JavaScript شاملة ومحسّنة مصممة لتوفير حل سهل الاستخدام لدمج بيانات المناطق والمدن والأحياء السعودية في أي تطبيق ويب. توفر هذه الأداة طريقة قوية ومرنة للتعامل مع البيانات الجغرافية الخاصة بالمملكة العربية السعودية، مما يجعلها مثالية للمطورين الذين يبنون تطبيقات تتطلب وظائف تعتمد على الموقع.

بفضل دعم CDN، وقدرات البحث المتقدمة، والتصميم المتجاوب، تهدف هذه الأداة إلى تبسيط عملية التطوير وتحسين تجربة المستخدم. سيرشدك هذا التوثيق خلال عملية تثبيت أداة المناطق السعودية المحسنة، وتكوينها، واستخدامها، مع تقديم أمثلة مفصلة وأفضل الممارسات.

2. المميزات

📊 بيانات شاملة

جميع المناطق السعودية الـ 13، وأكثر من 4500 مدينة، وأكثر من 3700 حي، مما يضمن مجموعة بيانات غنية للخدمات القائمة على الموقع.

🚀 دعم CDN

متاح عبر خدمات شهيرة مثل jsDelivr و unpkg للتكامل السريع وأوقات التحميل السريعة.

🎯 سهولة الاستخدام

إعداد الأداة بسطر واحد من التعليمات البرمجية، مثالي للتطوير والنشر السريع.

🔍 بحث متقدم

إمكانيات بحث وتصفية قوية تجعل من السهل العثور على مواقع محددة بسرعة.

🌐 دعم RTL

محسّن خصيصًا للغة العربية مع دعم من اليمين إلى اليسار لتجربة مستخدم طبيعية.

📱 تصميم متجاوب

واجهة مستخدم جميلة ومتكيفة تعمل بسلاسة عبر جميع الأجهزة وأحجام الشاشات.

3. التثبيت

دمج أداة المناطق السعودية المحسنة في مشروعك أمر بسيط. اختر من بين عدة طرق للتثبيت حسب احتياجات مشروعك وسير عمل التطوير.

عبر CDN (الطريقة الموصى بها)

الطريقة الأسرع والأكثر توصية لتضمين الأداة هي باستخدام شبكة توصيل المحتوى (CDN). ما عليك سوى إضافة العلامات التالية إلى HTML الخاص بك:

<!-- CSS -->
<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/YounisDany/saudi-regions-widget@main/dist/saudi-regions-widget.min.css">

<!-- JavaScript -->
<script src="https://cdn.jsdelivr.net/gh/YounisDany/saudi-regions-widget@main/dist/saudi-regions-widget.min.js"></script>

<!-- أو استخدم الملف المدمج (JS + CSS) -->
<script src="https://cdn.jsdelivr.net/gh/YounisDany/saudi-regions-widget@main/dist/saudi-regions-widget.bundle.min.js"></script>

عبر NPM

للمشاريع التي تستخدم Node.js ومدير حزم مثل npm، قم بتثبيت الأداة كاعتمادية:

npm install saudi-regions-widget-enhanced

تحميل مباشر

قم بتنزيل ملفات الأداة مباشرة من صفحة الإصدارات على GitHub وقم بتضمينها في مشروعك كأصول محلية.

4. الاستخدام

بمجرد التثبيت، يمكن دمج أداة المناطق السعودية المحسنة بسهولة في صفحة الويب الخاصة بك. هناك طريقتان أساسيتان لاستخدام الأداة: إعداد سريع للاحتياجات الأساسية وإعداد مفصل للتخصيص المتقدم.

الإعداد السريع

للنشر السريع، تتيح لك طريقة createQuick تهيئة الأداة بأقل قدر من التكوين:

<div id="saudi-widget"></div>

<script>
// إنشاء الأداة بسطر واحد من التعليمات البرمجية
SaudiRegionsWidget.createQuick("saudi-widget", {
    language: "ar", // 'ar' أو 'en'
    rtl: true // تعيين إلى true للعرض من اليمين إلى اليسار
});
</script>

الإعداد التفصيلي

لتحكم أكثر دقة في سلوك الأداة ومستويات البيانات ومعالجة الأحداث:

// إنشاء مثيل جديد
const widget = new SaudiRegionsWidget({
    dataLevel: "complete", // 'regions', 'regions-cities', 'complete'
    language: "ar", // 'ar' أو 'en'
    rtl: true,
    dataUrl: "https://cdn.jsdelivr.net/gh/YounisDany/saudi-regions-widget@main/dist/data/"
});

// انتظار تحميل البيانات
widget.on("ready", function() {
    // إنشاء قوائم متتالية (منطقة -> مدينة -> حي)
    widget.createCascadingSelects("widget-container");
});

// الاستماع للأحداث
widget.on("regionChanged", function(data) {
    console.log("تم اختيار منطقة:", data.region);
});

widget.on("cityChanged", function(data) {
    console.log("تم اختيار مدينة:", data.city);
});

widget.on("districtChanged", function(data) {
    console.log("تم اختيار حي:", data.district);
});

5. خيارات التكوين

يقبل مُنشئ SaudiRegionsWidget كائن خيارات يسمح لك بتخصيص جوانب مختلفة من سلوك الأداة ومظهرها:

الخيار النوع الافتراضي الوصف
dataLevel String 'complete' مستوى البيانات الجغرافية المراد تحميلها
language String 'en' لغة العرض ('ar' أو 'en')
rtl Boolean false اتجاه النص من اليمين إلى اليسار
searchEnabled Boolean true تفعيل وظيفة البحث

9. مستويات البيانات

توفر الأداة مستويات بيانات مختلفة لتحسين الأداء بناءً على احتياجات تطبيقك:

مستوى البيانات الحجم الوصف
regions ~2 كيلوبايت يحمل المناطق السعودية الـ 13 فقط
regions-cities ~85 كيلوبايت يحمل المناطق والمدن التابعة لها
complete ~1.2 ميجابايت يحمل المناطق والمدن وجميع الأحياء

11. أمثلة التطبيق

أداة المناطق السعودية المحسنة مستقلة عن الأطر ويمكن دمجها في أطر JavaScript المختلفة:

تكامل React

import { useEffect, useRef } from 'react';

function SaudiRegionsComponent() {
    const widgetRef = useRef(null);
    
    useEffect(() => {
        const widget = window.SaudiRegionsWidget.createQuick('saudi-widget', {
            language: 'ar',
            rtl: true
        });
        
        widget.on('regionChanged', (data) => {
            console.log('تم تغيير المنطقة:', data);
        });
        
        return () => widget.destroy('saudi-widget');
    }, []);
    
    return <div id="saudi-widget" ref={widgetRef}></div>;
}

تكامل Vue.js

<template>
    <div id="saudi-widget"></div>
</template>

<script>
export default {
    mounted() {
        this.widget = window.SaudiRegionsWidget.createQuick('saudi-widget', {
            language: 'ar',
            rtl: true
        });
        
        this.widget.on('regionChanged', this.handleRegionChange);
    },
    
    beforeDestroy() {
        if (this.widget) {
            this.widget.destroy('saudi-widget');
        }
    },
    
    methods: {
        handleRegionChange(data) {
            console.log('تم تغيير المنطقة:', data);
        }
    }
}
</script>
