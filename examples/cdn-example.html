الإعداد السريع

هذا المثال يوضح كيفية استخدام المكتبة مباشرة من CDN بدون تحميل أي ملفات:

<!-- CSS --> <link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/YounisDany/saudi-regions-widget@main/dist/saudi-regions-widget.min.css"> <!-- JavaScript --> <script src="https://cdn.jsdelivr.net/gh/YounisDany/saudi-regions-widget@main/dist/saudi-regions-widget.min.js"></script> <!-- أو استخدم الملف المدمج (JS + CSS) --> <script src="https://cdn.jsdelivr.net/gh/YounisDany/saudi-regions-widget@main/dist/saudi-regions-widget.bundle.min.js"></script>
جاري تحميل البيانات من CDN...
لم يتم اختيار أي عنصر بعد

الاستخدام السريع

يمكنك إنشاء المكتبة بسطر واحد فقط:

// الطريقة السريعة SaudiRegionsWidget.createQuick('widget-container', { language: 'ar', rtl: true }); // أو الطريقة التفصيلية const widget = new SaudiRegionsWidget({ dataLevel: 'complete', language: 'ar', rtl: true }); widget.on('ready', () => { widget.createCascadingSelects('widget-container'); });
